# Модель связи валютных курсов и цены на нефть

Проект по предмету "Оптимальное инвестирование и риск-менеджмент".

## Тема

Сравнительный анализ связи цены нефти Brent с валютными курсами Норвегии и Швеции.

## Основная идея

Норвегия рассматривается как экономика, чувствительная к нефтяному фактору.  
Швеция используется как сравнительная страна из того же региона, но без такой же нефтяной специализации.

## Гипотезы

1. NOK сильнее связан с Brent, чем SEK.
2. В кризисные периоды связь между Brent и NOK может усиливаться.
3. Нефтяной фактор может использоваться как индикатор валютного риска.

## Данные

Используются дневные ряды из FRED:

- DCOILBRENTEU - Brent
- DEXNOUS - USD/NOK
- DEXSDUS - USD/SEK

## Методы

- лог-доходности
- корреляции
- OLS с HAC-стандартными ошибками
- модель с лагами Brent
- rolling beta
- VaR
- Expected Shortfall
- backtesting VaR
- сценарный анализ нефтяного шока

## Как запустить

```bash
pip install -r requirements.txt
jupyter notebook oil_fx_risk_project.ipynb
```
